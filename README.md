# 📖 Project Overview

Brew Haven Café is a professional website designed for a local café to establish a strong online presence. The website showcases the café's menu, services, gallery, customer testimonials, and contact information through a modern and responsive interface.

---

# ✨ Features

- Responsive Navigation Bar
- Modern Hero Section
- About Us Section
- Food & Beverage Menu
- Image Gallery
- Customer Testimonials
- Contact Form
- Social Media Links
- Mobile Responsive Design
- Smooth Scrolling
- Interactive User Interface

# 🚀 Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Analthomas/FUTURE_FS_03.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

Or

Use **VS Code** with the **Live Server** extension for a better development experience.
